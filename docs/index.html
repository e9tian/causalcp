Diagnostic visualization for causal estimands

CP plots and local CP plots in a few lines of R.

CPplot is a plot-first R package for visualizing the relationship between estimated treatment effects and estimated propensity scores. It supports observational CP plots and local CP plots for instrumental-variable studies.

Start the tutorial View API reference

Installation

Install the development version from GitHub and load the package.

install.packages("remotes")
remotes::install_github("e9tian/CPplot")
library(CPplot)

Workflow

CPplot separates estimation from visualization. You can estimate the inputs with any model you like, then pass the resulting columns to the plotting functions.

Estimate inputs

Compute propensity scores and conditional treatment-effect summaries.

Draw the plot

Use cp_plot() or local_cp_plot() for diagnostic graphics.

Read slopes

Use the returned slope table as a compact covariance diagnostic.

What the diagnostic lines tell you

The CP plot is tied to the exact bracketing results in the paper. In the observational setting, the exact bracketing corollary shows that the signs of covariance terms involving the CATE tau(X) and the propensity score e(X) determine the ordering of ATE, ATT, ATC, and ATO. The three fitted lines in a CP plot estimate those covariance signs.

Read the slope signs as diagnostics. A positive fitted slope gives the implication listed below; a negative fitted slope reverses the corresponding inequality.

CP plot line Weight used Population analog Positive slope implies
Unweighted linear fit 1 cov{tau(X), e(X)} tau_ATT >= tau_ATE >= tau_ATC
Treated-tilted linear fit ehat_i cov{tau(X), e(X) | Z = 1} tau_ATT >= tau_ATO
Control-tilted linear fit 1 - ehat_i cov{tau(X), e(X) | Z = 0} tau_ATO >= tau_ATC

The local CP plot uses the same logic under the complier distribution. Replace the CATE by the conditional complier treatment effect tau^c(X), and use the estimated principal score pi_c_hat as the complier weight.

Local CP plot line Weight used Population analog Positive slope implies
Complier-weighted linear fit pi_c_hat_i cov{tau^c(X), e(X) | U = c} tau^c_ATT >= tau^c_ATE >= tau^c_ATC
Instrument-treated complier fit pi_c_hat_i * ehat_i cov{tau^c(X), e(X) | U = c, Z = 1} tau^c_ATT >= tau^c_ATO
Instrument-control complier fit pi_c_hat_i * (1 - ehat_i) cov{tau^c(X), e(X) | U = c, Z = 0} tau^c_ATO >= tau^c_ATC

Observational CP plot

In an observational study, the CP plot compares unit-level estimated CATEs tau_hat with estimated propensity scores ehat. The three fitted lines use full-sample weights 1, ehat, and 1 - ehat.

Minimal example

library(CPplot)

set.seed(1)
n <- 900
df <- data.frame(x1 = rnorm(n), x2 = rnorm(n))
df$ehat <- plogis(-0.30 + 1.05 * df$x1 - 0.45 * df$x2)
df$z <- rbinom(n, 1, df$ehat)
df$tau_hat <- -0.75 + 2.40 * df$ehat +
  5.20 * (df$ehat - 0.52)^2 +
  0.42 * df$x2 + rnorm(n, sd = 0.24)

fit <- cp_plot(
  df,
  ehat = "ehat",
  tau_hat = "tau_hat",
  treat = "z"
)

fit$plot
fit$slopes
Returned object Meaning
fit$plot The ggplot object, ready to print or customize.
fit$slopes Slopes, standard errors, and p-values for the three diagnostic fits.
fit$data_used The finite observations retained after filtering missing or nonfinite inputs.

Local CP plot for IV studies

In an IV study, replace the CATE by an estimated conditional complier treatment effect. Under the IV assumptions used in the paper, pi_c_hat can be the estimated conditional first stage.

Minimal example

library(CPplot)

set.seed(2)
n <- 900
df <- data.frame(x1 = rnorm(n), x2 = rnorm(n))
df$ehat <- plogis(-0.25 + 0.95 * df$x1 + 0.35 * df$x2)
df$z <- rbinom(n, 1, df$ehat)
df$pi_c_hat <- pmax(
  0.04,
  plogis(-0.35 + 1.05 * df$x1 - 0.95 * df$x2)
)
df$tau_c_hat <- -0.55 + 1.20 * df$ehat +
  5.60 * (df$ehat - 0.48)^2 +
  0.58 * df$x2 + rnorm(n, sd = 0.28)

fit <- local_cp_plot(
  df,
  ehat = "ehat",
  tau_c_hat = "tau_c_hat",
  pi_c_hat = "pi_c_hat",
  group = "z"
)

fit$plot
fit$slopes

Estimation helpers

CPplot includes simple transparent helpers for examples and quick checks. They use logistic regression for propensity scores and linear models for conditional contrasts. 

df2 <- estimate_cp_inputs(
  df,
  y = "y",
  z = "z",
  x = c("x1", "x2")
)

fit <- cp_plot(
  df2,
  ehat = "ehat",
  tau_hat = "tau_hat",
  treat = "z"
)

These helpers are optional. For applied work, you can estimate the inputs with causal forests, BART, SuperLearner, xgboost, or other models.

API reference

All plotting functions accept column names as strings. This makes it easy to use CPplot after estimating the inputs with any modeling workflow.

cp_plot()

cp_plot(data, ehat, tau_hat, treat = NULL,
        point_labels = c("Control", "Treated"),
        title = NULL, alpha = 0.45, point_size = 1)

Draw an observational CP plot and return the plot plus slope diagnostics.

ArgumentWhat to pass
dataA data frame containing the plotting inputs.
ehatName of the estimated propensity-score column, hat(e)(X).
tau_hatName of the estimated CATE column, hat(tau)(X).
treatOptional 0/1 treatment column used for point shapes and the point legend. The fitted lines still use the full sample with weights 1, ehat, and 1 - ehat.
point_labelsTwo labels for the 0 and 1 point groups. Default is Control and Treated.
titleOptional ggplot title.
alpha, point_sizePoint transparency and size.

Returns a CPplot_result list with plot, slopes, and data_used.

local_cp_plot()

local_cp_plot(data, ehat, tau_c_hat, pi_c_hat, group = NULL,
              point_labels = c("Control", "Treated"),
              title = NULL, alpha = 0.35, point_size = 0.8)

Draw a local CP plot for an IV study.

ArgumentWhat to pass
dataA data frame containing the local CP plotting inputs.
ehatName of the estimated IV propensity-score column, hat(e)(X).
tau_c_hatName of the estimated conditional complier treatment-effect column, hat(tau)^c(X).
pi_c_hatName of the estimated principal-score column. Under the IV assumptions, this is the estimated conditional first stage.
groupOptional 0/1 IV group column used for point shapes. The three fitted lines use all finite observations with weights pi_c_hat, pi_c_hat * ehat, and pi_c_hat * (1 - ehat).
point_labelsTwo labels for the 0 and 1 point groups. Default is Control and Treated.
title, alpha, point_sizeOptional plot title and point styling.

Returns a CPplot_result list with plot, slopes, and data_used. Rows with nonpositive or nonfinite pi_c_hat are dropped.

FunctionOutput
cp_slopes()A data frame with one row each for the unweighted, treated-weighted, and control-weighted linear fits.
local_cp_slopes()A data frame with one row each for the complier-weighted, treated-complier-weighted, and control-complier-weighted linear fits.

Each slope table contains fit, n, slope, std_error, and p_value.
